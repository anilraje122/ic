	5. Data binding
	
		The communication between HTML and js
			a. 1 way binding
			b. 2 way binding
			c. event based binding
			d. js to HTML (output element)
			e. js to HTML (input element)
		
		HTML has 2 types of elements
			a. input element : js can communicate with input element using "value" property
			b. output element : js can communicate with output element using "innerText" and "innerHTML" properties 
			
		Note : HTML tags are called elements in js
